Make yourself comfortable with vim

Being developer it looks quite fancy of modifying the text file using the command line tools such as vim,nano etc. But When i started working on command line text editors it was quite annoying at first. I tried multiple time to work with cli but duo to not having enough knowledge of shortcut keys i was like handicapped and have to go back to my old friends (Ui text editors) . In the local systems it was fine but More than often i had to search through server logs, modifying configuration files in remote servers. And while accessing those remote system you do not have such luxury and you have to use cli only. Then i decided to learn shortcut keys to make my life easier. after getting my hands dirty it was like charm working with these editors.

This post is walkthrough over some of the keys which might come handy while working with vim.

To start with just go to your terminal

For creating new file type vim yourFilename

if the file with the file name specified by you doesn’t exists a new file with the same name will be created, otherwise existing file will be opened.

in vim there are two modes of execution

  • normal mode : Here you can navigate through the file using shortcut keys, but you can not type anything
  • Insert mode : This is the mode where you can edit your file, but you can not apply any command.
  • Insert mode has following options
  • i to start inserting before the cursor pointer.
  • I to start inserting at the starting of the current line.
  • a to append before the cursor pointer.
  • A to append at the last of line

By default your file will be opened in normal mode.For switching to insert mode press ‘i’ . and for switching back to the normal mode press esc key.

For applying any command Make sure you are working in normal mode.

After adding some text in your file when you think you are done. go to normal mode and then type ‘:’ with following options

  • q! to quit without saving new changes.
  • w to saving the changes.File will still be open
  • :saveas filename (if you did not provided the file name already. i.e. If you opened vim using vim without file name)
  • wq! to saving the changes and then and quit. alternate of this command is (shift +zz key)

Now we will learn to navigate through file

Navigation

  • 0 move cursor to start of the current line.
  • $ move cursor to end of current line.
  • gg move cursor to start of the first line.
  • G move cursor to start of the last line.
  • G$ move cursor to end of the last line.
  • h move cursor to left of current position.
  • l move cursor to right of current position.
  • k move cursor to one line up.
  • j move cursor to one line down.
  • w move starting of the next word.
  • e move end of the next word.
  • b move starting of the previous word.
  • ) move one sentence forward.
  • ( move one sentence backward.
  • } move one paragraph forward.

  • { move one paragraph backward.

    Screen navigation

  • ctrl+f page down

  • ctrl+b page up

  • H move top of the screen.

  • M move middle of the screen.

  • L move bottom of the screen. #Insert

  • o this will add new line below the current line.

  • O this will add new line above the current line. Both of these command will automatically change the mode to insert mode.

    Delete

  • x delete character on cursor position.keep pressing x to remove characters on Right.

  • X delete character on left to cursor position

  • dd delete line.

  • dw delete one word these delete commands actually along with deleting copies the text. So you can past the last copied text using p

Note: use any number key to multiply the action by that number . i.e.

  • 3j to move cursor 3 lines up.

  • 5dw to delete 5 words.

    Repetition

    you can repeat any action by using .. i.e.

  • let say you added some text. press esc than . .text will be duplicated.

  • if your last command is 5x. it will delete 5 characters. then . will remove the next 5 characters.

    Replace character

    if you want to replace the character under cursor position,

  • r and the character you want to replace with.

  • R and the character you want to replace. this will replace the character and will take you to insert mode.

  • SS to substitute current line. #Redo/undo

  • u undo the last action.

  • ctrl+r to redo the last change

    search

Search is the most important action when working with the text file.

  • press f with the character to search . i.e. fp will search next p.
  • press F with the character to search. i.e. fp will search previous p.
  • press ***** to find next occurrence of current word.
  • press # to find previous occurrence of current word.
  • press / it will lead you to bottom of the terminal. then enter any word.it will take you to the first occurrence of that word.
  • for all of these command, press n to go to next character/word,press N to move to previous character/word #Move line
  • press : and then
  • **m ** and the number to move line up or down(-) by that number. i.e. m5 to move current line 5 line up. or -5 to move 5 line down.
  • m0 to move line before first line.
  • m$ to move line at the last.
  • 3,5m10 to move line number 3 to 5 after t10’th line.
  • .,5m 10 to move current line +next 4 lines after the 10’th line. * These are the basic command which will be most oftenly used while working with vim.practice will make you perfect and comfortable.
